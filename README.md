# COLACTRON INVEST

Plateforme privée de pilotage d'investissements immobiliers pour COLACTRON.

Objectif : centraliser le cycle complet d'un bien immobilier : recherche, analyse de rentabilité, visite, offre, achat, rénovation, location, revente, reporting et archivage.

La documentation produit et technique se trouve dans le dossier [`docs/`](./docs/README.md).
