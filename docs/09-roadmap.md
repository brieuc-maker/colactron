# 09 - Roadmap Produit

## V1 - Socle opérationnel

Objectif : disposer d'une interface utilisable par les associés pour centraliser les biens et suivre les décisions.

Fonctionnalités :

- Authentification.
- Dashboard.
- Gestion des biens.
- Fiche bien.
- Analyse financière manuelle.
- Pipeline Kanban.
- Documents.
- Visites.
- Historique.

## V2 - Pilotage complet

Objectif : gérer les travaux, finances et reporting.

Fonctionnalités :

- Module travaux complet.
- Module finances.
- Graphiques portefeuille.
- Carte interactive.
- Exports PDF et Excel.
- Alertes et notifications.
- Gestion location.
- Gestion vente.

## V3 - IA et automatisation

Objectif : transformer COLACTRON INVEST en copilote immobilier intelligent.

Fonctionnalités :

- Analyse IA des annonces.
- Scoring IA.
- OCR documents.
- Recherche documentaire intelligente.
- Génération offres d'achat.
- Résumé automatique des visites.
- Agent de veille immobilière quotidien.

## V4 - Intégrations avancées

Fonctionnalités :

- Immoweb / Zimmo / Biddit.
- Google Calendar.
- Google Maps.
- Odoo.
- Synchronisation bancaire.
- Emails.
- WhatsApp.

## V5 - SaaS potentiel

Objectif : rendre la plateforme multi-sociétés.

Fonctionnalités :

- Multi-tenant.
- Portefeuilles multiples.
- Permissions avancées.
- Abonnements.
- Onboarding client.
- Application mobile.
