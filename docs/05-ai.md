# 05 - Intelligence Artificielle

## Objectif

L'IA doit aider les associés à identifier, analyser et piloter les meilleures opportunités immobilières.

## Agents IA

### Agent Analyse Bien

Analyse une annonce et produit :

- Résumé.
- Points forts.
- Points faibles.
- Risques.
- Travaux probables.
- Prix cible.
- Valeur après rénovation.
- Loyer estimé.
- Score IA.
- Recommandation.

### Agent Rentabilité

Calcule :

- ROI.
- TRI.
- Cash-flow.
- Marge brute.
- Marge nette.
- Rendement locatif brut.
- Rendement locatif net.

### Agent Visite

Génère une checklist et transforme les notes de visite en compte rendu structuré.

### Agent Documents

OCR, classement automatique, résumé, recherche en langage naturel.

Exemples :

- "Montre-moi le compromis du bien de Villers-la-Ville".
- "Liste les factures travaux non payées".
- "Résume le PEB de ce bien".

### Agent Décision

Compare plusieurs biens et recommande les priorités.

## Score IA

Score sur 100 basé sur :

- Prix par rapport au marché.
- Potentiel de négociation.
- Potentiel de rénovation.
- Demande locative.
- Liquidité de revente.
- Risque travaux.
- Risque administratif.
- Qualité du quartier.
- Rentabilité estimée.

## Sorties attendues

Chaque analyse IA doit être structurée en JSON pour pouvoir être stockée et affichée dans l'interface.
